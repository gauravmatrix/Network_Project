🚀 Distributed Server Network with Traffic Optimization
📌 Overview

A Cisco Packet Tracer–based project that designs a scalable and secure network using VLANs, inter-VLAN routing, and distributed servers to optimize traffic and improve performance .

🎯 Features
VLAN-based network segmentation
Inter-VLAN routing (Router-on-a-Stick)
Trunking between switches
Distributed server setup
DMZ for security

🏗️ Network Structure
VLAN 10 → Client
VLAN 20 → DMZ
VLAN 30 → Server
VLAN 40 → Admin
⚙️ Tech Used
Cisco Packet Tracer
Routers, Switches, PCs, Servers

📊 Outcome
Reduced congestion
Better performance
Improved security
Scalable design
